# pytest-querys

一个基于 pytest 的服务查询集中管理工具。通过统一的配置和注册机制，将各种服务的查询管理器集中注册和管理，提供一致的接口来访问不同的后端服务。

## 简介

pytest-querys 是一个服务查询集中管理框架，它允许开发者通过统一的接口来管理和使用各种后端服务的查询功能。通过将服务对象集中注册到查询注册表中，您可以轻松地获取和使用不同的服务管理器，而无需关心底层的实现细节。

## 主要特性

- 🔧 **统一的查询接口** - 为所有服务提供一致的查询 API
- 📚 **集中式注册管理** - 所有服务查询管理器在统一注册表中管理
- 🔌 **易于扩展** - 支持自定义服务查询类型的开发和集成
- 🧩 **pytest 集成** - 深度集成到 pytest 测试框架中
- ⚡ **类型安全** - 提供完整的类型注解支持
- 🔍 **动态发现** - 自动发现和加载已注册的查询管理器
